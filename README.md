# Kasa

Ce projet a été réalisé dans le cadre du projet 5 de la formation Développeur Web d’OpenClassrooms.

Ce repository contient le front-end de l’application Kasa, une plateforme de location immobilière entre particuliers.  
L’application a été développée en React à partir de maquettes et de données fournies.

Le projet met en œuvre une navigation dynamique, des composants réutilisables et la gestion des routes.

## Technologies utilisées
- React
- JavaScript
- SCSS
- React Router
